# APE 2
Esta es una aplicación de consola desarrollada en Java para la gestión de datos de estudiantes matriculados en un curso. 
El objetivo principal del sistema es determinar el cálculo de la media de valores estructurados aplicando los principios 
de la Programación Orientada a Objetos (POO) y utilizando vectores para el almacenamiento de datos.

## 📋 Requisitos previos

Para ejecutar este proyecto, se necesitará:
* Un compilador de C++.

## 🚀 Instalación y Ejecución

1. Clonar este repositorio en la máquina local.
2. Abrir la carpeta del proyecto desde el IDE principal.
3. Compilar y ejecutar el archivo principal.

## ⚙️ Uso y Funcionalidades principales

El sistema permite gestionar un máximo de 20 estudiantes. Al iniciar, el gestor de personas presenta un menú interactivo.

* 1. Estudiantes: Permite visualizar un listado autonumérico de los alumnos registrados. A través de un submenú, puedes ingresar, modificar y eliminar
     los datos de la clase `Estudiante`.
* 2. Registro de calificaciones: Buscando por número de cédula, permite agregar, modificar o eliminar hasta un máximo de 7 notas por estudiante. 
* 3. Determinar el promedio de notas de un estudiante:* Muestra los datos personales y calcula el promedio de calificaciones de un alumno específico
  4. mediante su cédula.
* 4. Determinar el promedio de notas del curso: Calcula y muestra el promedio general de todas las notas registradas en el sistema.
